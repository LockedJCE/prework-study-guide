HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Unordered lists, ordered lists, and list items are ul, ol, and li
  • Src is important for images because it defines the location of the image
  • Alt is important to add on images as it helps with SEO and those on AT

    CSS

    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • A border indicates how much space around the padding and is solid and gray
    • The box-shadow adds shadow effects around an element's frame

    Git

    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it
    • git reset file-name: unstage a file while retaining the changes in working directory
    • git add file-name: add a file as it looks now to your next commit
    • git commit -m: commit your staged content as a new commit snapshot
    • git branch branch-name: create a new branch at the current commit
    • git merge branch-name: merge the specified branch's history into the current one

    JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • Curly brackets contain the code for the task we want to perform.
    • A function's name is followed by parentheses
